What is HTML?

HTML - Hyper Text Markup Language, is the standard markup language used for creating web pages and web applications.
It is not a programming lang but a markup lang that defines the structure of the content. HTML allows developeers to create and organize various types of content such as text, images, vedios and links.

Basic Structure of HTML document: it has HTML tags

<!DOCTYPE html> -- declaration of document type
<html> -- root element that encloses the entire html document
<head> -- this element contains meta data and any references to extrenal files and title of the document or any other elements that describe the document
<title>Title is given here</title> -- title of the document
</head> -- closing of head
<body> -- contains visible contentof the web page
Content of the web page --content
</body> -- end of body
</html> -- end of html doc
other html tags: tags are enclosed within < and >
h1 - h6 -- for different sizes of headings with h1 as highest level and h6 as lowest
p -- paragraph
a -- to create hyperlinks in html, it navigates between pages and websites
eg: <a href="url">link text </a>
url can be given or a simple # can be given if there is no target page to redirect. it can also have an attribute target="blank" which will redirect to a new page within the browser
img -- to embed images into webpage it has attributes src="url" to specify the location/path of the image source and also attributes of width and height and alt and title to specify the name of the image in times of image breakdown and when cursor points the image respectively. the src can either be absolute path(full length path i.e. from this pc) or a relative path (exact path), among which relative is highly prefered.
<img src="url" alt="description" title="imgTitle"/>
div -- used for grouping and structing other HTML elements
br -- to create a new line
!--....-- for comments. eg: <!-- this is a comment-->
span -- an inline container used to markup a part of text. when executed within html there wont be any difference. it will be helpful while applying styles using css or js to that particular tag elements

Importance of DOCTYPE declaration:

<!DOCTYPE> is not an HTML tag. It is an information to the browser about the type of document to expect. This ensures that the web page is parsed (resolve into its components parts and describe their syntactic role.) the same way by different web browsers.

Semantic Elements:

it clearly describes the meaning to both browser and the developer.
Non-semantic elements like <div> <span> dont specify anything about the content in it.
but Semantic elements like <form> , <table> , <article> defines the contents inside them.
semantic elements include:
<header> -- it represents the introductory content
<nav> -- it is used to define a section of navigation links
<main> -- this is similar to body
<section> -- it represents the content section, typically with a heading
<article> -- it represents a self contained section, such as a blog post or any news article etc
<footer> -- it represents the closing content of a web page/section

Attributes and meta data:

An attribute is a piece of markup lang used to adjust the behavior or dispaly of an HTML element, simply, to change the functionality of an element.
it will be mentioned inside inside a tag with specified value assigned to it.
eg: <tagName attributeName="value"><content>
Some attributes are:
id -- is used to uniquely identify of each and every element in a HTML files
class -- used to apply css styles to multiple elements with same class name
style -- used for css styling directly to an element, this includes color, font, size etc.
meta data -- this tag is placed within head section of an HTML doc. it provides info about the char encoding, description, keywords etc. to help search engines crawl and rank web pages.

Media Elements:

can access vedio and audio files using media elements
<vedio> -- to access embedded vedios
<audio>> -- to access embedded audio files
<iframe> -- to access embedded files from other resources

HTML New Elements:
